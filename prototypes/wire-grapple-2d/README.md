# Prototype: Wire Grapple 2D

## Hypothesis

ショクワンダー式ワイヤーアクション（振り子ではなく直線引き寄せ）は2D横スクロールで気持ちいいか？

## How to Run

1. Godotでプロジェクトを開く
2. `prototypes/wire-grapple-2d/test_level.tscn` を開く
3. **F6** で現在のシーンを実行

## Controls

| 操作 | 内容 |
|------|------|
| 左クリック | ワイヤー射出 |
| A / D | 左右移動 |
| Space | ジャンプ / 壁蹴り |

## Level Guide

- **茶色の地形**: ワイヤー付着可能（壁に到達すると張り付く）
- **灰色のブロック**（中央）: 付着不可（失敗フィードバック確認用）
- 射程外に撃つとワイヤーが伸びて戻るフィードバックが出る
