*Created: 2026-04-26*
*Status: Draft*

---

## Elevator Pitch

> ワイヤーを射出して目標地点へ直線的に引き寄せられる高速移動を軸にした2Dアクションプラットフォーマー。
> ワイヤーで縦横無尽に飛び回りながら敵を斬り倒し、エリアを制圧して先へ進め。

---

## Core Identity

- **Genre**: 2D Wire Action Platformer
- **Engine**: Godot 4.6 (GDScript)
- **Core Mechanic**: ワイヤー射出 → 直線引き寄せ移動 → 壁張り付き → 次アクション
- **Feeling**: スピーディー、慣性の気持ちよさ、スキル上達の手応え

---

## MDA Framework

### Mechanics
- ワイヤー射出（マウス方向・射程制限あり）
- 直線引き寄せ移動（移動中は衝突無効）
- 壁張り付き（垂直面のみ）
- しゃがみ（当たり判定縮小・壁張り付き解除）
- 通常ジャンプ・壁蹴りジャンプ

### Dynamics
- 連続スウィングによるスピード加速
- ワイヤー角度・タイミングを読んだルート選択
- エリア内の敵配置を読んだ立ち回りと制圧順序の選択

### Aesthetics
- 上達の快感（フロー状態）
- 敵を倒す爽快感・エリア制圧の達成感
- 慣性・物理感の心地よさ

---

## Player Fantasy

「クモの糸で空中を自在に飛び回るスーパーヒーロー」「忍者のワイヤー移動」

---

## Core Loop

```
ステージ開始
  → エリア突入（出口封鎖状態）
  → ワイヤーで移動しながら敵へ接近
  → 戦闘（斬撃・ワイヤー絡め）→ 敵を倒す
  → エリア内全敵撃破 → 次エリアへのルート開放
  → 次エリアへ移動 → 繰り返し
  → 最終エリアクリア → ステージクリア
```

---

## Technical Notes

- ワイヤー物理: 直線速度ベクトル制御（`collision_mask=0` で移動中の衝突無効化）
- プレイヤー: `CharacterBody2D` ベース
- ワイヤー描画: `Line2D` または `ShaderMaterial`
- カメラ: `Camera2D` + スムージング
