# Game Concept: VectorFlow

*Created: 2026-04-26*
*Status: Draft*

---

## Elevator Pitch

> ワイヤーを射出して目標地点へ直線的に引き寄せられる高速移動を軸にした2Dアクションプラットフォーマー。
> 壁から壁へと連続ワイヤーで飛び回り、複雑なステージを最速で駆け抜けろ。

---

## Core Identity

- **Genre**: 2D Wire Action Platformer
- **Engine**: Godot 4.6 (GDScript)
- **Core Mechanic**: ワイヤー射出 → 直線引き寄せ移動 → 壁張り付き → 次アクション
- **Feeling**: スピーディー、慣性の気持ちよさ、スキル上達の手応え

---

## MDA Framework

### Mechanics
- ワイヤー射出（マウス方向・射程制限あり）
- 直線引き寄せ移動（移動中は衝突無効）
- 壁張り付き（垂直面のみ）
- しゃがみ（当たり判定縮小・壁張り付き解除）
- 通常ジャンプ・壁蹴りジャンプ

### Dynamics
- 連続スウィングによるスピード加速
- ワイヤー角度・タイミングを読んだルート選択
- ステージ構造の理解による最短経路の発見

### Aesthetics
- 上達の快感（フロー状態）
- スピードランへの挑戦
- 慣性・物理感の心地よさ

---

## Player Fantasy

「クモの糸で空中を自在に飛び回るスーパーヒーロー」「忍者のワイヤー移動」

---

## Core Loop

```
ステージ開始
  → 地形を読む（どこにフックを打てるか）
  → ワイヤー射出 → スウィング
  → 次の足場/フックポイントへ
  → ゴール到達 → タイム記録
  → より速いルート模索 → リトライ
```

---

## Technical Notes

- ワイヤー物理: 直線速度ベクトル制御（`collision_mask=0` で移動中の衝突無効化）
- プレイヤー: `CharacterBody2D` ベース
- ワイヤー描画: `Line2D` または `ShaderMaterial`
- カメラ: `Camera2D` + スムージング
